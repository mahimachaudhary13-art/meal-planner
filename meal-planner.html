Cloud auto-sync — free, via GitHub

Auto-saves to your own private GitHub Gist a few seconds after every change, and pulls the latest whenever you open the app. One-time setup:
1. Create a free account at github.com
2. Settings → Developer settings → Personal access tokens (classic) → Generate new token → tick only the gist box → set expiry to "No expiration"
3. Paste the token below (do this once on each device)

The token lives only in this browser and can only touch gists — nothing else on your GitHub. If both devices edit at the same time, the most recent save wins.
